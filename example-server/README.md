# Example Paper Server using Docker

This stack exists to make it as easy as possible to set up a server with the database SebUtils needs.

## Deploying the example stack

Ensure you have Docker installed first. It's available on Windows, MacOS and Linux. You can download it here: https://www.docker.com/

1. Save the docker-compose.yml file where you want to launch your server
2. Next to the docker-compose.yml file, create the folders `minecraft/plugins`
3. Download the latest SebUtils release and copy it into `minecraft/plugins`
4. Run the server with `docker compose up -d`

You shouldn't need to edit any configurations at this point, unless you want to change the database password or the SebUtils features.

### Some useful commands:

`docker compose logs --follow` - show the logs from the server
`docker compose restart` - restart the server
`docker compose down` - stop the server and remove the containers (does not remove any data)
`docker compose up -d` - starts the server and leaves it running in the background

## Managing SebUtils

SebUtils has a config file, stored in `minecraft/plugins/SebUtils/config.json`

**Please do not edit this file when the server is running!**

If you need to edit the config whilst the server is running, use the web dashboard or the in-game commands. If you cannot access either one, you will need to stop the server, edit the config file, then restart the server.
